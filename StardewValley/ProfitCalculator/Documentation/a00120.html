Detailed Description

Class used to calculate the profits for crops. Contains all the settings for the calculator and the functions used to calculate the profits. Also contains the list of crops and the crop parsers. TotalCropProfit(Crop) and TotalCropProfitPerDay(Crop), TotalFertilizerCost(Crop), TotalFertilzerCostPerDay(Crop), TotalSeedsCost(Crop), TotalSeedsCostPerDay(Crop) are the main functions used to calculate the profits. RetrieveCropsAsOrderderList and RetrieveCropInfos are the main functions used to retrieve the list of crops and crop infos.
